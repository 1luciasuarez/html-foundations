Contenido

En esta página practicamos la jerarquía de headings y una estructura clara de secciones.

Estructura del documento

El documento tiene un H1 global y subsecciones organizadas con H2 y H3.

Doctype y lenguaje

El doctype activa el modo estándar. El atributo lang define el idioma del documento.

Metadatos básicos

Incluyen charset UTF-8, viewport para responsive, title y meta description.

Headings y jerarquía

Los headings comunican el esquema del contenido. No se usan para estilo.

Regla práctica

un H1, luego un H2 y debajo H3. Evitar saltos innecesarios de nivel.

Accesibilidad

Una jerarquía clara mejora la navegación con lectores de pantalla.
